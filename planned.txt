## рџ§© Trust Layer (CORE COMPONENT)

### Trust is computed using:

* Command abnormality (`acc_cmd`)
* Execution abnormality (`acc_exec`)
* Accumulation (`theta`)
* Memory (`omega`)
* Non-linear abnormality mapping (`alpha`)
* Weighted evidence (`w_cmd`, `w_exec`)

### Outputs:

* Trust score `T`
* Risk score `R = 1 - T`
* Decision:

  * `PASS` в†’ allow execution
  * `WARN` в†’ cautious
  * `BLOCK` в†’ stop machine
